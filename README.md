# Neobip Level Editor

Web-based level editor for [NeoShot](https://github.com/ChefJulio/neoshot).

**[Open the Editor](https://chefjulio.github.io/neobip-editor/)**

## Controls

| Key | Action |
|-----|--------|
| 1-7 | Tools: Brick, Erase, Select, Dropper, Multiply, Steel, Polygon |
| W/S | Brush height +/- |
| A/D | Brush width +/- |
| Q/E | Multiply value -/+ |
| , / . | Nudge hue -/+ 0.02 |
| R | Rotate (selection or stamp) |
| F | Flip (selection or stamp) |
| X | Toggle X mirror |
| Y | Toggle Y mirror |
| Tab | Toggle sized brick mode |
| Delete | Delete selection |
| Arrows | Move selection |
| Cmd+Z | Undo |
| Cmd+Shift+Z | Redo |
